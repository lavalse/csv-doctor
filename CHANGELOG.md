# Changelog

## v1.0.0 (2026-03-27)

初回リリース。

### 機能

- CSV ファイルの読み込み・自動クリーニング（エンコーディング / 区切り文字の自動検出、NFKC 正規化、不可視文字除去、空行除去、ヘッダー重複補完）
- クリーンな UTF-8 CSV としてダウンロード
- GeoJSON / CZML / KML エクスポート（座標列を UI から選択）
- 座標列の自動推定（英語 / 中国語 / 日本語ヘッダーに対応）
- ファイルサイズ上限 50 MB / 行数上限 100,000 行
- Shift_JIS / CP932 / UTF-8 (BOM あり・なし) 対応
